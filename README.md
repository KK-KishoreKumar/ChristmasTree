# Merry christmas and a happy new year!

Just a little bit of code I threw together christmas morning 2018. Give it a shot, and spread the Christmas cheer!

## Dependencies

### Ubuntu:
`libncurses5-dev`

`libncurses5w-dev`

### Arch:
`ncurses`

## Compiling

Run following: `gcc -o christmas christmas.c -lncurses`
